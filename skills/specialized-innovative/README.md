# 专精特新小巨人申报助手

专精特新小巨人申报助手是一个基于CowAgent框架的企业申报智能助手，帮助企业完成专精特新小巨人申报全流程。

## 功能特点

### 1. 多模式信息收集
- **文档上传**：支持PDF、Excel、Word等格式的企业文档智能提取
- **批量输入**：一次性提供企业关键信息，快速完成信息录入
- **对话引导**：通过智能问答逐步收集企业信息，体验友好

### 2. 智能资格评估
- **政策规则匹配**：基于专精特新政策要求进行自动评估
- **关键指标计算**：自动计算财务、研发、市场等关键指标
- **成功率估算**：基于企业数据估算申报成功率
- **改进建议**：提供针对性的改进建议

### 3. 申报材料生成
- **市场占有率说明**：生成1000字专业说明文档
- **企业介绍**：生成2000字企业详细介绍
- **申请书框架**：生成完整的申请书填写框架
- **佐证材料清单**：生成详细的材料准备清单

### 4. 材料完整性检查
- **智能匹配**：自动识别上传文件对应的材料类型
- **完整性评估**：计算材料完整性百分比
- **缺失清单**：生成详细的缺失材料清单
- **准备指南**：提供每项材料的具体准备指南

## 安装与配置

### 1. 前置依赖
```bash
# 安装Python依赖
pip install pdfplumber openpyxl python-docx
```

### 2. 技能安装
将本目录复制到CowAgent的skills目录下：
```bash
cp -r specialized-innovative /path/to/cowagent/skills/
```

### 3. 启用技能
在CowAgent配置文件中启用本技能：
```json
{
  "skills": {
    "specialized_innovative": {
      "enabled": true,
      "config": {
        "workflow": {
          "timeout": 3600,
          "max_sessions": 100
        }
      }
    }
  }
}
```

## 技术架构

### 模块设计
```
专精特新申报主Agent
    ├── 信息处理模块（InfoProcessor）
    │   ├── 文档提取（PDF/Excel/Word）
    │   ├── 文本解析（批量输入）
    │   └── 对话引导（智能问答）
    │
    ├── 评估计算模块（AssessmentCalculator）
    │   ├── 政策规则匹配
    │   ├── 关键指标计算
    │   ├── 资格评估
    │   └── 成功率估算
    │
    ├── 材料生成模块（MaterialGenerator）
    │   ├── 市场占有率说明
    │   ├── 企业介绍
    │   ├── 申请书框架
    │   └── 材料清单
    │
    └── 材料检查模块（ProofChecker）
        ├── 文件智能匹配
        ├── 完整性评估
        ├── 缺失清单生成
        └── 准备指南
```

### 数据流
```
用户输入 → 信息收集 → 数据清洗 → 资格评估 → 材料生成 → 材料检查 → 输出成果
    │          │           │          │           │           │
    └──文档上传└──文本解析─┘          └──规则匹配─└──模板填充─└──智能匹配
         │          │                       │           │           │
         └──对话引导└──批量输入             └──指标计算─└──AI生成──└──完整性评估
```

## 配置说明

### 技能配置（skill_config.json）
```json
{
  "name": "专精特新小巨人申报助手",
  "description": "帮助企业完成专精特新小巨人申报全流程的智能助手",
  "version": "1.0.0",
  "category": "企业服务",
  "enabled": true,
  "config": {
    "workflow": {
      "steps": ["信息收集", "资格评估", "材料生成", "材料检查", "完成输出"],
      "timeout": 3600,
      "max_sessions": 100
    },
    "assessment_rules": {
      "specialization": {
        "market_years": 3,
        "main_revenue_ratio": 70,
        "revenue_growth_2y": 5
      }
      // ... 更多规则配置
    }
  }
}
```

### 企业数据要求
**必需字段：**
- `company_name`：企业名称
- `establishment_year`：成立年份
- `industry`：所属行业
- `main_business`：主营业务
- `revenue_2023`：2023年营收（万元）
- `main_revenue_2023`：2023年主营业务收入（万元）
- `rd_expense_2023`：2023年研发费用（万元）
- `employee_count`：员工总数
- `rd_employee_count`：研发人员数
- `patent_count`：专利数量
- `main_products`：主要产品
- `market_share`：市场占有率（%）

**可选字段：**
- 近三年财务数据（2021-2023）
- 资产负债数据
- 客户与认证信息
- 研发机构信息

## 常见问题

### Q1: 技能无法启动
**A:** 检查CowAgent配置文件中技能是否启用，以及依赖包是否安装。

### Q2: 文档提取失败
**A:** 确保已安装 `pdfplumber`、`openpyxl`、`python-docx` 等依赖包。

### Q3: 评估结果不准确
**A:** 检查企业数据是否完整，或更新 `assessment_rules` 配置。

### Q4: 材料生成内容不专业
**A:** 可以连接大模型API，在生成方法中调用AI生成更专业的内容。

### Q5: 文件匹配不准确
**A:** 在 `proof_checker.py` 中调整材料的关键词匹配规则。
