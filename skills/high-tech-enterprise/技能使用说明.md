# 高新技术企业申报技能使用说明

## 一、技能概述

**高新技术企业申报技能**是一个专门帮助企业完成高新技术企业（高企）申报全流程的智能助手。该技能基于国家《高新技术企业认定管理办法》，集成了8大条件评估、创新能力评分、申报成功率预估、材料清单生成等功能，为企业提供一站式的申报指导服务。

## 二、执行逻辑（工作流程）

### 第1步：信息收集 → 第2步：资格评估 → 第3步：材料生成 → 第4步：材料检查 → 第5步：完成输出

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  信息收集   │───▶│  资格评估   │───▶│  材料生成   │───▶│  材料检查   │───▶│  完成输出   │
│ 收集企业信息│    │ 评估8大条件 │    │ 生成材料清单│    │ 检查材料完整│    │ 生成最终报告│
│ 财务数据等  │    │ 计算创新能力│    │ 提供模板    │    │ 识别缺失材料│    │ 创建专属文件夹│
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

## 三、按顺序完成的功能

### 1. 信息收集阶段
**功能：** 收集企业基本信息、财务数据、知识产权信息、研发情况
**具体完成：**
- ✅ 企业基本信息解析（名称、成立时间、所属领域）
- ✅ 财务数据收集（销售收入、研发费用占比、高品收入占比）
- ✅ 知识产权统计（发明专利、实用新型专利、软件著作权）
- ✅ 人员情况收集（职工总数、科技人员占比、研发人员数）
- ✅ 支持三种输入方式：上传文档、批量输入、对话引导

### 2. 资格评估阶段
**功能：** 评估企业是否符合高企申报条件
**具体完成：**
- ✅ **8大条件评估**（逐一检查）：
  1. 企业注册成立一年以上
  2. 拥有核心自主知识产权
  3. 技术属于国家重点支持领域
  4. 科技人员占比不低于10%
  5. 研发费用占比符合要求
  6. 高品收入占比不低于60%
  7. 创新能力评价71分以上
  8. 无重大安全环保事故

- ✅ **创新能力评分计算**（100分制）：
  - 核心自主知识产权（30分）：数量、先进性、支持作用、获得方式
  - 科技成果转化能力（30分）：根据知识产权数量评定转化等级
  - 研究开发组织管理水平（20分）：根据企业规模和研发管理情况
  - 企业成长性（20分）：根据研发投入和科技人员占比

- ✅ **申报成功率预估**：
  - 基础成功率：50%
  - 8大条件通过率加成：最多30%
  - 创新能力得分加成：最多20%
  - 总成功率范围：0-100%

### 3. 材料生成阶段
**功能：** 生成完整的高企申报材料清单
**具体完成：**
- ✅ **基础材料**（3项）：
  - 《高新技术企业认定申请书》（在线打印并签名+公章）
  - 证明事项告知承诺书
  - 企业营业执照复印件

- ✅ **知识产权材料**（2项）：
  - 知识产权证书及反映技术水平的证明材料
  - 参与制定标准情况材料（如有）

- ✅ **技术创新材料**（3项）：
  - 科技成果转化总体情况与转化形式说明
  - 科研项目立项证明及验收报告
  - 研究开发组织管理水平说明材料

- ✅ **产品服务材料**（2项）：
  - 高新技术产品（服务）关键技术说明
  - 相关生产批文、认证证书、产品质量检验报告

- ✅ **财务人员材料**（4项）：
  - 近三年研究开发费用专项审计报告（必需）
  - 近一年高新技术产品（服务）收入专项审计报告（必需）
  - 企业职工和科技人员情况说明材料
  - 近三年财务会计报告及纳税申报表

### 4. 材料检查阶段
**功能：** 检查佐证材料完整性，识别缺失材料
**具体完成：**
- ✅ 材料完整性检查（百分比显示）
- ✅ 已有材料清单展示
- ✅ 缺失材料清单识别
- ✅ 补充建议提供
- ✅ 材料检查报告生成

### 5. 完成输出阶段
**功能：** 生成最终评估报告和申报材料包
**具体完成：**
- ✅ **评估报告生成**：
  - 8大条件评估结果
  - 创新能力评分详情
  - 申报成功率预估
  - 改进建议

- ✅ **专属申报文件夹创建**：
  ```
  高企申报_公司名称_时间戳/
  ├── 公司基本信息.txt          # 公司基本信息
  ├── 文件夹结构说明.txt        # 文件夹结构说明
  ├── 原始文档/                 # 上传的原始文档
  ├── 评估报告/                 # 资格评估相关文件
  │   └── 资格评估报告.txt      # 详细评估报告
  ├── 生成材料/                 # 生成的申报材料
  │   └── 申报材料清单.txt      # 完整材料清单
  ├── 佐证材料/                 # 材料检查报告
  │   └── 材料检查报告.txt      # 材料完整性检查
  └── 最终申报包/               # 最终申报材料包
      └── 申报包说明.txt        # 申报包使用说明
  ```

## 四、使用方法

### 方式一：上传压缩包（推荐）
**特点：** 由于Web端不支持直接上传文件夹，用户可以将所有申报材料打包成ZIP压缩包上传，Agent自动解压并分析所有文件
**步骤：**
1. 输入"高企申报"开始流程
2. 选择"1.上传压缩包"选项
3. 上传包含企业申报材料的ZIP压缩包
4. Agent自动解压压缩包，扫描所有文件，分析文件类型和内容
5. 系统从文件中提取企业信息，并显示提取结果

**压缩包制作方法：**
```
1. 将所有申报材料放入一个文件夹
2. 右键点击文件夹选择"压缩"或"添加到压缩文件"
3. 生成ZIP文件（如：企业申报材料.zip）
4. 在Web端上传该ZIP文件
```

**推荐压缩包内容：**
```
企业申报材料.zip
├── 财务文件/
│   ├── 审计报告.pdf
│   ├── 财务报表.xlsx
│   └── 研发费用明细.xlsx
├── 法律文件/
│   ├── 营业执照.jpg
│   └── 资质证书.pdf
├── 知识产权/
│   ├── 发明专利证书.pdf
│   ├── 实用新型专利证书.pdf
│   └── 软件著作权证书.pdf
├── 技术文件/
│   ├── 研发项目立项报告.docx
│   └── 技术成果说明.pdf
└── 人员文件/
    ├── 员工花名册.xlsx
    └── 社保缴纳证明.pdf
```

**Agent处理流程：**
```
上传ZIP压缩包 → 自动解压 → 扫描所有文件 → 分析文件类型 → 提取企业信息 → 评估高企资格
```

### 方式二：上传文档
**特点：** 上传单个或多个文档，Agent从文档中提取信息
**步骤：**
1. 输入"高企申报"开始流程
2. 选择"2.上传文档"选项
3. 上传企业相关文档（审计报告、营业执照、专利清单等）
4. 系统从文档中提取信息
5. 补充缺失信息后完成评估

### 方式三：批量输入
**特点：** 一次性提供企业关键信息
**步骤：**
1. 输入"高企申报"开始流程
2. 选择"3.批量输入"选项
3. 输入企业信息，格式如：
   ```
   企业名称：测试科技有限公司
   成立时间：2020-05-10
   所属高新技术领域：先进制造与自动化
   上年度销售收入：3500
   近三年研发费用占比：5.2
   高品收入占比：68.3
   科技人员占比：12.5
   职工总数：120
   研发人员数：15
   发明专利数量：3
   实用新型专利数量：12
   软件著作权数量：5
   ```

### 方式四：对话引导
**特点：** 通过问答逐步提供信息
**步骤：**
1. 输入"高企申报"开始流程
2. 选择"4.对话引导"选项
3. 按照系统提示逐步提供企业信息
4. 系统将自动评估并生成报告

## 七、注意事项

### 1. 数据准确性
- 请确保提供的信息准确无误
- 财务数据需与审计报告一致
- 知识产权需有有效证书

### 2. 时间要求
- 申报需在每年规定时间内提交
- 材料准备需提前3-6个月
- 评估结果有效期6个月

### 3. 材料要求
- 所有材料需加盖公章
- 审计报告需由有资质机构出具
- 知识产权需在有效期内

### 4. 后续跟进
- 申报后需关注评审进度
- 可能需要补充材料或答辩
- 通过后需按时提交年报
